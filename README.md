Create a simple Ecommerce website.
In which we can add catagories and products from admin side and display it into client side.
In admin side we can add, Edit, Remove and details of the products and categories.
In client side there is a home page and all product page.
In all product page we can display all the products which is added by the admin.Also we can display catagory wise product list.
In home page we can display all the catagories and products lists.when we click on perticular catagories then it's display categories wise products lists.
We used identity user for login and registration.
When we click on add to cart button then products will be added in the cart.
In cart page we can see all the add to cart item.
If we directly going to the cart page by the browser url link then it's redirect to the login page first. Means first you need to login for seeing cart items.
We can change quntity of the product from cart page.
In cart page we can display products wise total also.
If there are multiple products in the cart then display product wise total and all products total also.
